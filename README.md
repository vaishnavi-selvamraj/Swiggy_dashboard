# Swiggy Data Analytics 

# 1. Business Scenario
Swiggy is one of India's largest online food delivery platforms connecting customers, restaurants, and delivery partners across multiple cities.
Every day, thousands of orders are placed through the platform. This generates large volumes of operational data related to:
•	Customers
•	Restaurants
•	Orders
•	Delivery partners
•	Payments and delivery performance
Swiggy wants to use this data to analyze business performance, improve delivery efficiency, and understand customer behavior.

# 2. Project Objective
The objective of this project is to build a complete data analytics pipeline that:
1.	Stores raw Swiggy datasets in a Lakehouse
2.	Cleans and validates the data using SQL
3.	Models the data inside a Data Warehouse
4.	Creates a Power BI Semantic Model
5.	Builds interactive Power BI dashboards
6.	Publishes the final report for business stakeholders

# 3. Power BI Report
Create an interactive Power BI dashboard that provides insights for business users.
The report should be published inside the Fabric workspace.

# Tools for using 
1. CSV (data set)
2. Excel
3. SQL (Data cleaning)
4. Power BI (Data Visualization)
